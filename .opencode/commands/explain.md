설명 대상: $ARGUMENTS

인자가 없으면 현재 프로젝트의 전체 구조와 아키텍처를 설명합니다.

사용 예시:
- `/explain` - 프로젝트 전체 구조 설명
- `/explain Publisher` - Publisher 인터페이스 설명
- `/explain backpressure` - Backpressure 개념 설명
- `/explain src/main/.../ArrayPublisher.java` - 특정 파일 설명

Reactive Streams 개념은 비유와 다이어그램으로 쉽게 설명합니다.
